# Monokai Theme

Tema Monokai clásico para el editor de código de DEX STUDIO.

## Características

- Esquema de colores Monokai fiel al original
- Soporte completo para syntax highlighting (keywords, strings, comments, funciones, etc.)
- Se integra con el selector de temas del editor

## Colores

| Elemento | Color |
|----------|-------|
| Keywords | `#f92672` (rosa) |
| Strings | `#e6db74` (amarillo) |
| Comments | `#75715e` (gris) |
| Functions | `#a6e22e` (verde) |
| Builtins | `#66d9ef` (cyan) |
| Numbers | `#ae81ff` (púrpura) |

## Uso

Una vez instalada, selecciona "Monokai" en el selector de temas del editor (menú de configuración).
