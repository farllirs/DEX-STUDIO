# File Icons

Iconos coloridos por tipo de archivo en el explorador de DEX STUDIO.

## Características

- Iconos específicos por extensión de archivo (`.py`, `.js`, `.html`, `.css`, `.json`, etc.)
- Colores distintivos para cada tipo de archivo
- Actualización automática al navegar el árbol de archivos
- Soporte para archivos de imagen, código, texto y configuración

## Tipos soportados

| Extensión | Color | Icono |
|-----------|-------|-------|
| `.py` | Azul | file-code |
| `.js` | Amarillo | file-code |
| `.html` | Naranja | file-code |
| `.css` | Púrpura | file-code |
| `.json` | Dorado | file-json |
| `.md` | Azul oscuro | file-text |
| `.png/.jpg/.gif` | Violeta | file-image |
| `.sh` | Verde | file-terminal |

## Uso

La extensión se activa automáticamente al instalarse. Los iconos del explorador de archivos se actualizan para reflejar el tipo de cada archivo.
