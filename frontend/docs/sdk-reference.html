SDK Reference

DEX STUDIO SDK

Referencia oficial para crear extensiones y usar APIs de DEX de forma estable.

Inicio rápido 
DEX.registerExtension({
  id: 'mi-extension',
  name: 'Mi Extension',
  version: '1.0.0',
  description: 'Describe la funcion principal'
}, {
  onInit: function() {
    DEX.log('Extension cargada');
  }
});
// Dex code successful

Tipos de extensión

editor

Editor, comandos, productividad, análisis.

theme

Temas visuales de la interfaz principal.

ui-theme

Diseño de layout y variables UI completas.

language

Extiende soporte de lenguajes/formatos.

Themes

Para temas UI usa manifest.json con category: "theme" o type: "ui-theme" y define variables en theme.css.

{
  "id": "mi-tema",
  "name": "Mi Tema",
  "version": "1.0.0",
  "category": "theme",
  "type": "ui-theme"
}

Recomendado: mantener IDs únicos y semver consistente.

Git y GitHub para Principiantes

Guía visual para usar Git en DEX STUDIO sin tocar terminal.

Ruta rápida recomendada

Primero configura tu perfil en Configuración > Git (token, nombre y correo). Luego entra a Git Workspace para inicializar/clonar, hacer commit y push.

1. Configura tu perfil

Ve a Configuración > Git y guarda:

Token GitHub
Nombre
Correo
2. Prepara un repositorio

En Git Workspace elige:

Inicializar actual
o Clonar repo
3. Revisa cambios

Usa la lista de cambios para hacer Stage, Unstage o Descartar.

4. Commit

Escribe mensaje claro y presiona Commit.

5. Conecta remoto

Pega URL en Remote URL (origin) y pulsa Set Remote.

6. Push

Presiona Push. DEX STUDIO aplicará tu nombre/correo configurados antes de subir.

Checklist de errores comunes
  • Si el push falla por permisos: revisa que el token tenga acceso al repositorio.
  • Si el commit sale con autor incorrecto: vuelve a guardar nombre/correo en Configuración > Git.
  • Si no ves repositorios: abre o clona uno dentro de tu carpeta de proyectos.
